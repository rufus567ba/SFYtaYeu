# 项目简介

## 核心功能
- 基于YOLOv8的乐高实例分割检测系统
- 支持图片、视频及摄像头进行检测
- 支持选择不同模型
- 实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并直接查看所有检测结果
- 支持检测结果保存
- 可自定义置信度、IOU阈值
- 可选择显示标签和原图

## 适用场景
- 乐高玩具的自动识别与分类
- 玩具库存管理
- 儿童玩具识别应用
- 自动化玩具分拣系统

## 技术特点
- 使用Pycharm+Anaconda或VSCode+Anaconda开发环境
- 基于Python 3.9、OpenCV-Python、PyQt5、Ultralytics、PyTorch 1.9等技术栈
- 包含3000张标注数据集，专注于Lego一类识别
- 提供完整的源代码、UI界面源文件、图标、数据集图片和项目运行教程
- 已训练好模型，配置成功环境可直接使用
